MkniXPost

Post-processor for MkniX .mec result files

Download .zip Download .tar.gz View on GitHub

Overview

MkniXPost is a GUI cross-platform post processor for intrepreting MkniX analyses. MkniX is a code design for the simulation of nonlinear flexible multibody systems, including thermal effects.

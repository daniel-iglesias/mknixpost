MkniXPost

Post-processor for MkniX analysis files

Download .zip Download .tar.gz View on GitHub

Overview

MkniXPost is a GUI cross-platform post processor for intrepreting MkniX analyses. MkniX is a code design for the simulation of nonlinear flexible multibody systems, including thermal effects.

Characteristics

  • Timeline control.
  • 3D graphical window with interactive mouse manipulation.
  • Rendered surfaces or wire representation of bodies.
  • Maps of stress and temperature fields.
  • 2D graphs plots of any result (DoF, energy, reactions, etc).
  • Image and video capture.
  • Stereographic rendering for depth sensing.
  • Built on top of cross-platform toolkits (Qt, VTK, gnuplot).
